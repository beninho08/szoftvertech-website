H3/4 – Felhasználói dokumentáció

Projekt: Magasság-alapú színházi jegyfoglaló rendszer (FastAPI)

1. Áttekintés

A rendszer lehetővé teszi, hogy a felhasználó magasságának megadásával foglaljon ülőhelyet, ezzel minimalizálva a takarásból adódó élményromlást.

2. Telepítés / elérés

# fejlesztői futtatás
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -e .[dev]
python -m app.main  # Swagger: http://localhost:8000/docs

3. Használati útmutató (Swaggeren keresztül)

LépésMűvelet
1Nyisd meg a Swagger UI-t: /docs
2Futtasd a GET /seats végpontot – ellenőrizd az ülésállapotot
3Használj POST /reserve → add meg: sor, oszlop, magassag_cm
4Siker esetén: 201 Created, foglalás rögzül
5Magasabb ülő miatti elutasítás: 409 Conflict + üzenet
6Foglalás törlése: DELETE /reserve/{id}

4. Hibaelhárítás

ProblémaOkMegoldás
409 ConflictElőtted magasabb foglalás vanVálassz másik sort vagy hátrébb ülést
404 Not FoundFoglalás ID nem létezikKérd le újra az üléslistát
422 Unprocessable EntityInput hibás (nem létező sor/oszlop)Ellenőrizd az értékeket Swaggerben
